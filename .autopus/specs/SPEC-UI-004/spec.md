# SPEC-UI-004: 사용자 개입(Human-in-the-Loop) 승인 시스템

## 1. 개요
에이전트 간의 연쇄 작업 중 사용자가 중간 결과물을 검토하고 승인하거나 반려할 수 있는 '품질 게이트'를 UI에 구현한다.

## 2. 요구사항 (EARS)
- **WHEN** 연쇄 협업 중 한 에이전트가 응답을 완료하면, **THE SYSTEM SHALL** 즉시 다음 단계로 넘어가지 않고 '승인 대기' 상태로 전환해야 한다.
- **WHILE** 승인 대기 상태인 동안, **THE SYSTEM SHALL** 결과 보고서 팝업 하단에 [승인 및 다음 단계 진행] 버튼과 [반려 및 재지시] 버튼을 보여주어야 한다.
- **WHEN** 사용자가 [승인]을 클릭하면, **THE SYSTEM SHALL** 그 결과값을 다음 에이전트의 입력값으로 전달하며 워크플로우를 재개해야 한다.
- **THE SYSTEM SHALL** 상단바에 현재 저장 중인 파일명(`.autopus/workflows/state.json`)을 명시해야 한다.

## 3. 워크플로우 제어권
- **AI**: 분석 및 결과 생성
- **Human**: 결과 검토 및 바통 터치 결정
